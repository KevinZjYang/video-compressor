# 视频压缩器-K

一个简单易用的视频压缩和裁剪桌面应用。

## 功能特性

- **视频压缩** - 支持多种压缩预设，满足不同场景需求
- **视频裁剪** - 精确裁剪视频片段
- **批量处理** - 支持批量压缩多个视频文件
- **中文界面** - 简洁友好的中文用户界面

## 技术栈

- 前端：Vue 3 + TypeScript + Element Plus
- 后端：Tauri 2 + Rust
- 视频处理：FFmpeg

## 环境要求

- Windows 10/11 (64位)
- WebView2 运行时（Windows 11 已预装）

## 开发

```bash
# 安装依赖
npm install

# 启动开发模式
npm run tauri dev

# 构建安装包
npm run tauri build
```
